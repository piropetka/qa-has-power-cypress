# QA-Has-Power-Cypress

## What is needed:

1. Node.js installed
2. Cypress

   ```bash
   $ npm install

   ```

## Usage:

1. Run tests:

   ```bash
   $ npx cypress open
   $ npx cypress run --headed --no-exit
   ```

   Or alternatively use one of commands:

   ```bash
   $ (npm bin)/cypress open
   $ ./node_modules/.bin/cypress open
   ```
